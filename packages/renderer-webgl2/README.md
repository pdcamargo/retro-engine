# @retro-engine/renderer-webgl2

WebGL2 implementation of the Retro Engine renderer HAL.

**Stub.** Every method throws `Error("WebGL2 backend is not implemented yet.")`. The package exists so the contract surface stays in view and consumer code can resolve its types.

Implementation tracked in [`docs/roadmap/webgl2-backend.md`](../../docs/roadmap/webgl2-backend.md).
